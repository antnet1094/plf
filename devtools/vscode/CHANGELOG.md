# Changelog

All notable changes to the "plf-syntax" extension will be documented in this file.

## [1.0.1] - 2026-03-18
- Enhanced branding with official icon and high-impact README.
- Better short description for Marketplace visibility.

## [1.0.0] - 2026-03-18
- Initial release of PLF Syntax Highlighting.
- Support for @meta, @role, @context, @tools, @rules, @fallback, @chain, @task, @output.
- Template variables highlighting `{{ var }}`.
- Language configuration (comments, brackets).
