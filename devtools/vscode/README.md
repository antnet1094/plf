# PLF Syntax Highlighting

Esta extensión proporciona resaltado de sintaxis para el formato **Prompt Language Format (.plf)**.

## Características
- Resaltado de secciones `@meta`, `@rules`, `@context`, etc.
- Detección de directivas de control (`NEVER`, `ALWAYS`, `IF`, `MAX`, `MIN`).
- Resaltado de variables de template `{{ variable }}`.

## Uso
Simplemente abre cualquier archivo con extensión `.plf` y el resaltado se activará automáticamente.
